프로그램이란? 데이터 + 기능(함수) => 명령

--------------------------
#.자료의 명세 (데이터 찾기, 변수찾기)
입력(데이터)변수: 사용자가 입력한 가위,바위,보 : input 
처리(데이터)변수: 컴퓨터가 임의로 낸 수 : computer
출력(데이터)변수: 사용자승패: result

#.처리과정
1.반복 (승패가 날때까지, 무승부면 반복)
    1.1 사용자가 가위바위보를 낸다
    1.2 컴퓨터가 가위바위보를 임의로 정한다
    (1.3 사용자와 컴퓨터가 내 수를 판단한다)

2.사용자의 승패를 판단한다
    2.1 사용자가 가위일때
        -컴퓨터가 가위이면 무승부
        -컴퓨터가 바위이면 사용자 패
        -컴퓨터가 보이면 사용자 승
    2.2 사용자가 바위일때
        -컴퓨터가 가위이면 사용자 승
        -컴퓨터가 바위이면 무승부
        -컴퓨터가 보이면 사용자 패 
    2.3 사용자가 보일때
        -컴퓨터가 가위이면 사용자 패
        -컴퓨터가 바위이면 사용자 승
        -컴퓨터가 보이면 무승부

3.사용자의 승패 출력
